2007

Founding of Loruki

Loruki was founded with the goal of providing a platform that simplifies the deployment and scaling of web applications. It started as a Ruby-focused platform

2010

Acquisition by Salesforce

Salesforce.com acquired Loruki, bringing financial stability and resources to the platform

2011

Expansion Beyond Ruby

Loruki expanded its support beyond Ruby by adding support for other programming languages like Node.js and Clojure

2012

Introduction of Add-ons Marketplace

The Loruki Add-ons marketplace was introduced, allowing developers to easily integrate various services into their applications

2014

Support for Docker Containers

Loruki announced support for Docker containers, allowing developers to run any language or framework on the platform

2018

Loruki Review Apps

Loruki announced Loruki Review Apps, enabling developers to create disposable test apps for each pull request

2019

Introduction of Private Spaces

Loruki introduced Private Spaces, a network-isolated environment for running applications

2020

Launch of Loruki Teams

Loruki launched Loruki Teams, providing collaboration features for development teams
